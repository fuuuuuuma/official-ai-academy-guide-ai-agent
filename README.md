# 公式AI無料講座ガイド｜AIエージェントチャンネル\n\n> OpenAI / Anthropic / Google / Cursor / Manus の公式アカデミーを、実際の講座画面と一次情報に基づいて一社ずつ案内する公開用リサーチ資料です。\n\n## このリポジトリの成果物\n\n- [スライド](./slides.html) — 撮影・解説用。全30枚、各アカデミー章は公式リンクと実スクリーンショットから始まります。\n- [1枚ガイド](./onepager.html) — 推奨講座、学べること、向く人を一覧できるWeb資料です。\n- [プレゼント](./present.html) — 5社リンク集、選び方、コピー可能な学習メモです。\n- [画像一覧](./img/) — 上記の中に埋め込んだ公式サイト実スクリーンショットです。\n\n## この資料の方針\n\n業務設計・調査・コンテンツ制作・実行型AI活用の観点を最初に置きます。 ただし、特定の製品を推す比較資料ではなく、5社を一社ずつ学習の入口として解説します。\n\nこの資料は「OpenAIとAnthropicの比較」ではありません。各サービスに無料または無料で始められる公式学習リソースがあり、目的・前提・進め方が異なります。したがって、動画も資料も **1社を説明し切ってから次へ進む** 構成です。各章は次の順番を守ります。\n\n1. 公式アカデミーのリンクを最初に表示する。\n2. 実際の公式画面を見せる。\n3. 具体的な推奨講座の直リンクを出す。\n4. そこで学ぶ内容を、日本語で噛み砕く。\n5. 向く人・前提・最初の一歩・受講前の注意を示す。\n\n英語ページについては、公開されているコース概要、目次、対象者、必要環境を読み取り、日本語で要約しています。ログイン後にしか見られない動画本文・字幕・配布資料を、見えていないのに「文字起こしした」とは記載していません。動画がある講座は、公開ページで確認できる講義数・時間・カリキュラムを明示し、受講後に確認すべき実践ポイントへ訳しています。\n\n## 受講前に必ず確認すること\n\n- 「無料」はサービス全体の全コンテンツが無条件で無料、という意味ではありません。アカウント、無料クレジット、地域、機能アクセス、受講枠が関わる場合があります。\n- Google Skillsはコースによって表示・クレジット条件が異なります。資料では「No cost」表示を確認できるハンズオンを補足していますが、開始画面の条件を確認してください。\n- Manusは日本語の基礎コース画面にサインインが表示されます。利用枠、料金、機能条件をアカウント側で確認してから進めてください。\n- OpenAI Academy、Anthropic Academy、Cursor Learnも、UI・カリキュラム・受講条件は更新され得ます。公開直前と受講直前にリンクを再確認してください。\n\n## 1. OpenAI Academy\n\n単発の依頼から、文脈・出力・境界・レビューを備えた再現可能なワークフローへ進む公式講座です。\n\n**推奨講座：** [Agents and Workflows](https://academy.openai.com/public/courses/agents-and-workflows-bieml)\n\n**公式アカデミー：** [academy.openai.com](https://academy.openai.com/)\n\n### 学べること\n\n- 文脈を渡す設計\n- 出力形式と境界の決め方\n- レビューと改善の回し方\n- 繰り返し業務をワークフロー化する考え方\n\n### 公開情報から読める講座の構成\n\n公式ページでは約90分・セルフペース・中級者向けとして案内。AI Foundations と Applied AI Foundations の次に、普段の仕事を一つ持ち込んで学ぶ構成です。\n\n### おすすめする人と、開始後の行動\n\nChatGPTを単発利用で終わらせず、毎週の調査・資料作成・確認作業を「設計して任せる」側に進みたい人。\n\n開始したら、ホームの AI Foundations を確認し、次にこの講座で「毎週くり返す1業務」を題材にする。\n\n**受講前の注意：** 受講にはChatGPTアカウントと、エージェント機能にアクセスできる環境が案内されています。\n\n## 2.
