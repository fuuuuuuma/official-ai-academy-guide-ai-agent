OFFICIAL AI LEARNING GUIDE / 2026.07

公式AI講座を
1社ずつ、正しく始める。

AIエージェントチャンネルの視点で、仕事の設計・実行までつながる5社の公式学習コースを整理。 すべて公式リンクと実スクリーンショット付きです。

01

5つの公式アカデミー

「どちらが上か」ではなく、何を学べるか・誰が先に開くべきかを、各サービス単位で確認してください。

OpenAI Academy

Agents and Workflows

単発の依頼から、文脈・出力・境界・レビューを備えた再現可能なワークフローへ進む公式講座です。

公式アカデミー ↗
推奨講座を開く ↗

  • 文脈を渡す設計
  • 出力形式と境界の決め方
  • レビューと改善の回し方
おすすめする人：ChatGPTを単発利用で終わらせず、毎週の調査・資料作成・確認作業を「設計して任せる」側に進みたい人。

最初の一歩：ホームの AI Foundations を確認し、次にこの講座で「毎週くり返す1業務」を題材にする。

受講にはChatGPTアカウントと、エージェント機能にアクセスできる環境が案内されています。

Anthropic Academy

Claude Code in Action

Claude Codeを、ファイル・コマンド・コンテキスト・外部連携まで一続きの開発作業として学ぶ公式無料コースです。

公式アカデミー ↗
推奨講座を開く ↗

  • ファイル操作・コマンド実行・コード理解
  • CLAUDE.md・/init・@参照によるコンテキスト管理
  • カスタムコマンドとPlan / Thinkingの使い分け
おすすめする人：CLIを触れる、または触れるようになりたい人。Claude Codeを「チャット」ではなく、自分のリポジトリで動く作業環境として使いたい人。

最初の一歩：Skilljarの受講登録後、セットアップとコンテキストの章を見て、手元の小さなリポジトリで再現する。

公式案内ではCLI経験、Claude CodeとAPIキーが前提です。内容閲覧にはSkilljarアカウントが必要です。

Google Skills

初級: 生成 AI の概要

Google Skillsの初級パスで、生成AI・LLM・責任あるAIの基礎を、公式の短いアクティビティ単位で整理します。

公式アカデミー ↗
推奨講座を開く ↗

  • 生成AIと大規模言語モデルの基礎
  • 責任あるAIの考え方
  • クラウド上のAIサービスの位置づけ
おすすめする人：GeminiやVertex AIを使う前に、用語と責任ある利用の土台を短い単位で作りたい人。クラウド学習が初めての人。

最初の一歩：日本語表示のままこの初級パスを開始し、理解があいまいな用語を一つずつメモする。実装する人は次にGen AI SDKの無料ハンズオンへ。

Google Skillsはコンテンツにより受講条件が異なります。無料クレジットやNo cost表示など、開始画面の条件を必ず確認してください。

Cursor Learn

Working with agents（Cursor Learn）

Cursorの公式Learnは、AIの基礎からエージェントを使った機能実装・バグ修正・テストまで、開発者の実務に沿って学べる教材群です。

公式アカデミー ↗
推奨講座を開く ↗

  • モデル、幻覚、トークン、コンテキストの基本
  • ツール呼び出しとエージェントの仕組み
  • コードベースを理解させる方法
おすすめする人：AI搭載エディタを便利な補完で終わらせず、既存コードを読み、変更を検証する共同作業者として使いたい開発者。

最初の一歩：Cursor Learn の Working with agents から開始し、必ず小さな変更で差分とテストを自分で確認する。

CursorのUI・モデル・料金は変わり得ます。学習中も公式Learnの最新表示を基準にしてください。

Manus Academy

Manus - 基礎

Manusが自分のコンピュータで調査・操作し、ドキュメントやスライドなどの成果物へ進む仕組みを、基礎から体験する公式コースです。

公式アカデミー ↗
推奨講座を開く ↗

  • チャットボットと実行型エージェントの違い
  • チャット・QC・AIモード・テンプレート・ファイル・リプレイの見方
  • 許可を伴うブラウジングと並列リサーチ
おすすめする人：プログラミングを前提にせず、調査・企画・資料作成などの仕事を、完成物まで進めるAIの使い方として学びたい人。

最初の一歩：サインイン後に基礎コースを開始し、最初は「調査→成果物→確認」の一往復だけを小さなテーマで体験する。

開始画面にサインインが表示されます。利用枠・料金・機能条件はアカウント画面で確認してから始めてください。
